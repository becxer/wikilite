## What is wikilite
 * Markdown viewer
 * Auto following GitURL
 * It's like multi-repo Jekyll

## How to install

 1) pre-required

        $ pip install Flask mistune tqdm

 2) Install wikilite

        $ git clone https://github.com/becxer/wikilite.git
        $ cd wikilite
        $ python wikilite-server.py 5007

 3) Access to http://localhost:5007
